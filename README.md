# 🏥 Health Insurance Payment Prediction

A Machine Learning web application that predicts health insurance claim amounts based on user details.

## 📌 Features Used
- Age  
- Gender  
- BMI  
- Blood Pressure  
- Diabetic  
- Number of Children  
- Smoker  

## 🤖 Models Trained
- Linear Regression  
- Polynomial Regression  
- Random Forest  
- Support Vector Regressor  
- XGBoost  

🏆 **XGBoost achieved the best performance and was selected for deployment.**

## 🚀 Web App

The trained model is deployed using **Streamlit**.  
Users can enter their details and instantly get an estimated insurance payment.

To run locally:

```bash
streamlit run app.py
